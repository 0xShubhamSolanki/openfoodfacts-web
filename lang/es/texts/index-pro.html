Características claves

Datos validados

Asegúrate que los datos de tu producto sean completos y estén actualizados. Tus datos se convertirán en la fuente de referencia una vez que sean públicos. Y como sabemos que pueden ocurrir pequeños errores, te informaremos de forma privada cualquier problema de calidad de datos que detectemos antes de la importación, para que puedas corregirlo.

Obtenga sugerencias para mejorar fácilmente tu Nutri-Score

  • Mejora significativamente el Nutri-Score con una reducción razonable de azúcar, sal o ácidos grasos saturados.
  • Detecta productos con un perfil nutricional más bajo que productos similares en el mercado.

Compatible con cualquier XLSX/ODT que puedas tener

Si bien proporcionamos una plantilla fácil de usar una vez que te registras, puedes reutilizar cualquier archivo existente, en cualquier formato. The platform will allow you to match your column names with the Open Food Facts format.

Compatible con EQUADIS, AGENA3000 (y pronto más)

Si eres cliente de EQUADIS o AGENA 3000, puedes enviar tus datos automáticamente a Open Food Facts sin cargo adicional. Please get in touch if you're using a different system.

Ayudándote con las puntuaciones

Open Food Facts has computed Nutri-Score since it was the "5-color score" in 2015, and is now computing the Eco-Score.

Adopta fácilmente el Nutri-Score

Nutri-Score se calcula automáticamente a partir de los datos nutricionales.

  • Comprende el cálculo de Nutri-Score con todos sus componentes.
  • Verifica el Nutri-Score si ya lo ha calculado.

Calcula con precisión el Eco-Score de tus productos

El Eco-Score se calcula a partir de una categoría precisa, información sobre las etiquetas, los ingredientes y sus orígenes y componentes de embalaje.

  • Asegúrate de obtener el mejor Eco-Score posible eliminando aquelos errores vinculados a la información faltante.
  • Importing your products on the public database will allow you to benchmark them with best practises.

Obtén explicaciones detalladas de puntuación sea para Nutri-Score y Eco-Score

Una vez que sus datos son importados, proporcionamos explicaciones detalladas, incluyendo advertencias sobre los datos que potencialmente estén ausentes.

Compartir una vez, obtén datos armonizados sea en diversos lugares

Una vez que configures una importación automática, tus productos se sincronizarán automáticamente todas las noches. De lo contrario, la plataforma memoriza su formato de archivo para que las actualizaciones sean muy fáciles.
